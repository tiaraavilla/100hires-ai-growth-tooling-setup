## Project Overview

This repository documents the first step of my portfolio project for the **100Hires Junior Growth Marketing Specialist** application process.

The task was to install and configure AI supported work tools, create a public GitHub repository, and document the setup process clearly.

Instead of treating this as a basic technical checklist, I approached it as a small **growth operations onboarding project**. My goal was to show how I learn new tools, structure information, solve blockers, and communicate progress in a way that is clear, repeatable, and useful for future work.

---

## Positioning Statement

As a growth marketing candidate, I see this setup as more than a tool installation task.

It reflects three capabilities that are important in modern growth teams:

| Capability | Why It Matters |
|---|---|
| Learning agility | Growth roles often require adapting quickly to new tools, channels, and workflows |
| Systems thinking | Marketing execution improves when processes are documented and repeatable |
| AI literacy | AI tools can support research, experimentation, documentation, and productivity |

This repository is my first step in building a public portfolio that shows not only what I completed, but also **how I think and work**.

1. Market and competitor research  
2. Customer segment analysis  
3. Campaign ideation  
4. Landing page and messaging experiments  
5. Outreach workflow documentation  
6. Marketing operations and process improvement
