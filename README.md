## Issues I Ran Into and How I Solved Them

### Issue 1: Searching in the wrong place

At first, I searched for Claude Code through the Cursor website marketplace, but the result I needed did not appear.

**How I solved it:**  
I realized that the correct place to install the extension was inside the Cursor desktop app. After opening the Extensions section in Cursor, I was able to find the relevant Claude Code extension.

---

### Issue 2: Choosing the correct Claude Code extension

When searching for Claude Code, I found several extensions with similar names.

**How I solved it:**  
I checked the publisher and selected the extension associated with Anthropic instead of choosing a third party extension. This helped me avoid installing the wrong tool.

---

### Issue 3: Understanding GitHub access

When connecting Cursor to GitHub, I had to review the repository access permissions and understand what access I was giving.

**How I solved it:**  
I reviewed the permission screen, confirmed that the connection was for Cursor, and continued so I could complete the portfolio setup workflow.

---

### Issue 4: Making the README more useful than a checklist

The instruction only required a basic README, but I wanted the file to show how I think, learn, and communicate.

**How I solved it:**  
I structured this README as a setup log that includes the purpose of the task, tools installed, steps completed, issues encountered, solutions, and key learnings.
